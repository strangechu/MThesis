創造情報学専攻 修士論文 標準フォーマット 2009年度版

このページで，2009年版の東大情報理工 創造情報の修士論文標準フォーマットを配布しています． 標準フォーマットといっても，サンプルを提供しているだけですが， これらを参考に修士論文を作成してください． このフォーマットに合わせた LaTeX のスタイルファイルも置いてあります．

以下の点に注意してください．

  1. 表紙等，定型のフォーマットを必要とする部分は， できる限りこのサンプルに合わせてください．
  2. このフォーマットは様々な記述を想定していますので， 各自の論文では不要な部分もあるでしょう．適宜，選択あるいは省略してください．
  3. サンプルには現れない種類の記述は，各自で工夫してください．
  4. 今後さらに多くの種類の記述に対応できるよう充実させていく予定です．
  5. 論文をLaTeXで作成する場合には，下記のスタイルファイルを使ってください． それ以外の場合には，サンプルに近い出力になるよう，各自で設定してください．

2010年1月現在，博士論文のサポートはしていません． スタイルファイルでオプション doctor を指定するとなにか出てきますが，おそらく，その内容は適切なものではありません．

スタイルファイルについて

黒木祐介さんが作成した， 工学部計数工学科数理情報工学コースの卒業論文用クラスファイル suribt をもとに， 創造情報学専攻用に必要な修正を加えています． また，コンピュータ科学専攻の修士・博士論文用スタイルファイルの内容も参考にしました．

LaTeXスタイルファイル使用前の準備

日本語LaTeXのソースファイルが処理できるような環境を整えておいてください． また，奥村晴彦さんの作成しているクラスファイル jsbook.cls が必要です． (jsbookには，英文環境(englishオプションを指定)で，List of figures や List of tables の最初の行が不揃いになるという問題がありましたが，最新版では修正されています．-- 2009.01.26記．) オリジナルの配布元からも 入手できますし，以下のパッケージには付属しています．

当方では，以下のパッケージを用いて動作を確認しています．

  • linux: 土村 展之さんの ptetex3
  • MS Windows: 阿部紀行さんの TeXインストーラ3

ダウンロード

  • linux用: cimt.tar.gz
  • MS Windows 用: cimt.zip

現在のところ，linux 用と MS Windows 用のスタイルファイルを用意しています．

「linux用」は，日本語エンコーディングを EUC-JP に，行末を LF にしています． 「MS Windows用」は，日本語エンコーディングを Shift_JIS に，行末を CR LF にしています． どの組み合わせが最適かは，OS，TeXシステム，エディタなどに依存します． ご自分の環境に合わせて，必要ならば，変換を行ってください．

なお，MacOS X 上で，「MS Windows 用」スタイルファイルを用いて論文を作成できたとの報告をいただいています．

インストール

ここではインストール方法の概略を述べます． 詳細については，配布ファイル中に含まれているマニュアル (cimt.pdf) を参照してください．

まず，以下の手順を実行して， 上記サンプル1と同様のファイルが生成されることを確認してください．

  1. 配布ファイルを展開する．
  2. ディレクトリ class に移って，platex cimt.insを実行する．カレントディレクトリに cimt.cls ができる．
  3. cimt.cls をディレクトリ sampleJ にコピーする．
  4. ディレクトリ sampleJ に移って，以下のコマンドを実行する: 
        platex sampleJ
    jbibtex sampleJ
    platex sampleJ
    platex sampleJ
英語で書く場合には，sampleE ディレクトリで，同様の方法で sampleE.tex から サンプル2 が作れることを確認してください． また，sampleEP.tex から サンプル3 が作れます．
